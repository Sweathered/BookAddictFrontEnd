Register

Registers a user based upon the supplied fname, lname, email, password

Response: 

{
	registered: (boolean), whether registration was successful
	email: (String), whether the email passed was valid or invalid
}

Auth

Authenticates a user based upon the supplied email and password

Response 

{
"validUser" : (boolean) whether the user is valid or not
"userDetails" : {
	"fname" : (string) first name of user,
	"lname" : (string) last name of user,
	"apiKey" : (string) api key,
	"userType" : (string) type: Limited or Premium,
	"isRestricted" : (string) n or y
	}
}

Profiles

Gets profile information for a given user

Response 

{
	"fname": (string) first name of user,
	"lname": (string) last name of user,
	"apiKey": (string) apikey,
	"userType": (string) type: Limited or Premium,
	"email": (string) email of user
}

changePassword

Changes password for a user

Response 

{
  "success" : boolean depending on success of operation
}
